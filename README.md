# Minecraft Server Selector (C++)

A simple, lightweight C++ console application that allows users to pick a Minecraft server from a text-based menu. 

## 🚀 Features
* Clean console menu interface
* Real-time keyboard input capturing using `std::cin`
* Dynamic path selection using `if`/`else if`/`else` control flow logic

## 🛠️ Concepts Learned
* Working with basic standard input/output streams (`<iostream>`)
* Initialising and manipulating integer variables
* Managing syntax layout and resolving compiler bracket alignment errors

## 💻 How to Run
1. Compile the code using any standard C++ compiler (like GCC or Clang):
   ```bash
   g++ main.cpp -o ServerSelector
   ```
2. Run the executable:
   ```bash
   ./ServerSelector
   ```
